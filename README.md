# Sisyphus Type (In-Development)
A brief typing game made for my Software Development course, Sisyphus Type lets you type 
the Myth of Sisyphus from Camus's essay up slanted "terrain." A concept exploration, the 
slanted text gives a feeling of movement without compromising one's reading comprehension.

## How to Run
From a virtual environment, launch app.py as with "python3 app.py" and play in your browser.
